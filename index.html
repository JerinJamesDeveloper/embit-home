The Story Behind Embit

Indie Developer & Flutter Architect

I've built 12 Flutter apps in the last 3 years. MVPs for startups, SaaS platforms, client projects. Every single time, I spent the first week setting up the same architecture: Clean Architecture, BLoC, dependency injection, auth flows, RBAC. The same folder structure. The same boilerplate.

After project #8, I had enough. I automated everything I was repeating. Built starter kits. Created a CLI that generates features in seconds. What used to take 3 days now takes 10 minutes.

The Vision: Every Flutter developer should spend their time building features users love, not fighting with folder structures and boilerplate. Embit is the toolkit I wish existed when I started.

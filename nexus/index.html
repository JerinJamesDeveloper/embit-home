Architecture

The architecture inside.

Enterprise structure that survives growth. Every decision made for production workflows. 

lib/
├─ core/
│  ├─ config/          → dev / staging / prod
│  ├─ network/         → API client + interceptors
│  │  └─ interceptors/
│  │     ├─ auth_interceptor.dart
│  │     ├─ retry_interceptor.dart
│  │     └─ error_interceptor.dart
│  ├─ error/            → Exception → Failure mapping
│  ├─ router/           → Route guards + RBAC
│  ├─ theme/            → Dark / light system
│  ├─ di/               → Dependency injection
│  └─ ai/               → AI layer (Pro+)
│     ├─ openai_provider.dart
│     └─ mock_ai_provider.dart
├─ features/
│  ├─ auth/             → Complete auth module
│  ├─ profile/          → Profile module (Pro+)
│  ├─ payment/          → Payment module
│  │  ├─ gateways/
│  │  │  ├─ stripe_gateway.dart
│  │  │  └─ razorpay_gateway.dart
│  └─ [your_features]/   → CLI-generated
└─ shared/              → Reusable components

Commands

Powerful commands, simple syntax.

Every command is designed to be memorable, composable, and fast to type.

# Feature module (full 3-layer scaffold) embit feature -n <name> [--nav-bar] # Data model + optional BLoC state embit model -f <feature> -n <name> [--with-state] # Use case (get | post | stream) embit usecase -f <feature> -n <name> -t <type> # Repository + data source embit repo -f <feature> -n <name> # List all available commands embit --help ✓ All commands auto-register in DI + routes.
📁

embit feature

Generates the full three-layer structure (data / domain / presentation) with BLoC, repository interface, and route entry in one shot.

📄

embit model

Creates an entity and its data model. Add --with-state to scaffold BLoC state and event classes automatically.

🧩

embit usecase

Generates a typed use case with the correct abstract class signature, failure handling, and DI registration wired in.

🗄️

embit repo

Scaffolds the repository interface in domain and its implementation in data, then registers both in GetIt automatically.
